# Service Fabric Mesh Sample Application Model Templates

Following is the list of examples that shows how to achieve a particular scenario using Service Fabric Application Model v2. 

|Example Name|Scenario Description|
|------------|--------------------|
| [Hello World Mesh](./helloworld) | Learn about how to deploy a single container microservices application and connect to the service endpoint. |
| [Private  Registry](./prvate-registry) | Learn about how to use container images from a private registry. |
| [Azure File Volume](./azurefiles-volume) | Learn about how to store data on Azure Files. |
| [Votingapp](./votingapp) | Learn about how to create an application with a frontend and backend service that uses DNS-based resolution. |
| [Visual Objects](./visualobjects) | Learn about how to independently scale and upgrade microservices within an application.  |
